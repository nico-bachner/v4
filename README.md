## Run Locally

Install [pnpm](https://pnpm.io)

```sh
npm i -g pnpm
```

Clone the project

```sh
git clone https://github.com/nico-bachner/v4.git
```

Follow the remaining steps

```sh
cd v4 # Go to the project directory
pnpm i # Install dependencies
pnpm dev # Start the dev server
```
